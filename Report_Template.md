## 2. Task A：3-α 温度敏感性指数（23分）

- 你实现的 $rate_3alpha(T)$ 与 $sensitivity_nu(T0,h)$ 核心思路：
- 本任务实现恒星3-α反应率的温度敏感性分析，核心步骤如下：
- 1. 实现 rate_3alpha(T) 函数
     功能：计算3-α反应率的温度相关部分 q(T)
     公式： $q(T) = 5.09 * 10^11 * (T8)^(-3) * exp(-44.027 / T8)$ 其中 $T8 = T / 10^8$
     功能说明：输入开尔文温度T，计算归一化温度T8，代入公式得到反应率q(T)；并添加温度合法性校验（T>0），避免数值溢出。
  2. 实现 sensitivity_nu(T0, h) 函数
     功能：计算参考温度T0处的温度敏感性指数 nu
     定义：nu = (T/q) * (dq/dT)
     方法：采用前向差分法近似计算导数 dq/dT
     步长规则：绝对步长 $delta_T = h * T0$ (默认 $h=10^-8$ )
     功能说明：先获取T0处的反应率q0，再通过差分法计算导数，最终根据定义得到nu。
  3. 实现 nu_table(T_values, h) 函数
     功能：批量计算多个温度点的敏感性指数
     功能说明：遍历温度列表，对每个温度调用sensitivity_nu函数，返回 (温度, nu) 列表。 
- 使用的温度点与步长：
- 使用的温度点：1.0e8 K, 2.5e8 K, 5.0e8 K, 1.0e9 K, 2.5e9 K, 5.0e9 K
- 使用的步长参数：相对步长 h = 10^-8 
- 计算结果表：

| T0 (K) | ν(T0) |
|---:|---:|
| 1.0e8 | 41.03 |
| 2.5e8 | 14.61 |
| 5.0e8 | 5.81 |
| 1.0e9 | 1.40 |

- 物理解释：在低温与高温区，ν 的变化说明了什么？
- 1. 低温区（T0 约 $10^8$ K）
nu 值约为 41.03，数值显著偏高。这说明在恒星氦燃烧的初始阶段，3-α反应率对温度变化极度敏感。
物理意义：温度发生微小的相对变化，反应率会被放大数十倍；这意味着恒星核心仅需轻微升温，就会触发剧烈的氦闪现象，是恒星结构稳定性的关键影响因素。
2. 高温区（ $T0 > 10^9$ K）
nu 值随温度升高持续下降，甚至变为负值。
物理意义：高温下，反应率的指数项 $exp(-44.027 / T8)$ 的温度依赖大幅减弱，反应率增长趋势放缓；当温度过高时，反应率甚至可能随温度升高而下降，导致系统对温度的敏感性大幅降低，恒星进入稳定的氦燃烧后期阶段。
3. 整体规律
nu 随温度升高单调下降，清晰反映了3-α反应从“点火极易触发”到“燃烧稳定”的物理演化过程。这一量化结果是分析恒星主序后演化、氦闪机制等核心天体物理问题的重要基础。
